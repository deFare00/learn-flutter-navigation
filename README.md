# 📖 Overview

This repository is part of my **Flutter Learning Journey**, documenting my progress from learning the fundamentals to building complete mobile applications.

On **Day 1**, I focused on understanding how Flutter builds user interfaces using widgets, creating responsive layouts, and implementing navigation between multiple screens.

---

# 🎯 Learning Objectives

* Understand Flutter project structure
* Learn the Widget Tree concept
* Build UI using StatelessWidget and StatefulWidget
* Create responsive layouts
* Navigate between multiple screens
* Pass data between pages
* Return data from another page
* Organize Flutter projects using a clean folder structure
